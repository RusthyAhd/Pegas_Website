# Pegas Company Website

This is the official website for Pegas company, built with modern web technologies for optimal performance and SEO.

## Features

- Responsive design that works on all devices
- Modern, professional layout
- SEO optimized with meta tags
- Contact form functionality
- Smooth scrolling navigation
- Mobile-friendly hamburger menu
- Fast loading and lightweight

## Structure

- `index.html` - Main website file
- `styles.css` - All styling and responsive design
- `script.js` - Interactive functionality
- `sitemap.xml` - SEO sitemap for search engines
- `robots.txt` - Search engine crawling instructions

## Customization

You can easily customize this website by:

1. **Content**: Edit the text in `index.html`
2. **Colors**: Modify the color variables in `styles.css`
3. **Images**: Add your logo and images to the images folder
4. **Services**: Update the services section with your offerings
5. **Contact Info**: Update contact details in the contact section

## Deployment

This website can be deployed to any static hosting service:

1. **Netlify** (Recommended)
2. **Vercel**
3. **GitHub Pages**
4. **Firebase Hosting**
